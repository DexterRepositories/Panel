How to use YPanel Script
Note: Use Ubuntu 20.04

1. Get DB Informations
-DB_Host:
-DB_User:
-DB_Pass:
-DB_Name:
2. Ilagay ung DB Information sa 3 Category (Premium, VIP, Private)
3. Para sa OpenVPN, nasa mismong Script ang DB Information, same lang din ang need palitan
4. Eto ung script:
wget -O zoey.sh "Link ng script mo dito" -q; bash zoey.sh
Example kung nasa github
wget -O zoey.sh https://github.com/freenetsapinas/sshscript/blob/main/zoey.sh -q; bash zoey.sh
5. Change mo ung mga files na nasa loob ng script, iupload kung saan at itugma para gumana ng maayos ang script
